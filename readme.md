# BookWise Backend

Backend API for the BookWise application built using Node.js, Express.js, and MongoDB.

---

## 🚀 Features

- REST API support
- MongoDB database connection
- Express.js server
- Environment variable support
- Ready for Render deployment

---

## 📁 Project Structure

```bash
bookwise_submission/
│
├── src/
│   └── server.js
│
├── package.json
├── package-lock.json
├── .gitignore
├── README.md
└── .env
````

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* dotenv

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Nirbhayjakhar3196/Practise-module-5-backend.git
```

Go to project folder:

```bash
cd bookwise_submission
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Locally

Start the server:

```bash
npm start
```

OR (if using nodemon)

```bash
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file in the root directory and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

---
